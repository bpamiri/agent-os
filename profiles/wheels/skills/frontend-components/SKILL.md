# Frontend Components

## When to use this skill:

- When creating reusable UI components (buttons, cards, badges, alerts)
- When building form components (inputs, selects, checkboxes, radio buttons)
- When implementing navigation components (navbar, sidebar, breadcrumbs, tabs)
- When creating modal dialogs, dropdowns, or tooltip components
- When working on .cfm partial files in views/ directories
- When building table components with sorting, filtering, or pagination
- When implementing component variants (primary, secondary, danger, etc.)
- When adding interactive states (hover, focus, active, disabled)
- When creating layout components (containers, grids, cards)
- When using DaisyUI component classes or Tailwind utilities

This Skill provides Claude Code with specific guidance on how to adhere to coding standards as they relate to how it should handle frontend components.
