# Frontend DaisyUI

## When to use this skill:

- When using DaisyUI component classes (btn, card, input, modal, etc.)
- When implementing DaisyUI button variants (btn-primary, btn-secondary, btn-ghost)
- When building forms with DaisyUI form components (form-control, label, input)
- When creating navigation with navbar, menu, tabs, or breadcrumbs
- When implementing modals, drawers, or dropdown components
- When using DaisyUI alert, badge, or indicator components
- When applying DaisyUI theme colors (primary, secondary, accent, neutral)
- When working on .cfm view files that use DaisyUI classes
- When implementing responsive DaisyUI layouts
- When customizing DaisyUI themes in tailwind.config.js

This Skill provides Claude Code with specific guidance on how to adhere to coding standards as they relate to how it should handle frontend DaisyUI.
