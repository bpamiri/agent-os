# Frontend Responsive

## When to use this skill:

- When implementing mobile-first responsive designs
- When adding Tailwind breakpoint prefixes (sm:, md:, lg:, xl:, 2xl:)
- When creating layouts that adapt from mobile to desktop
- When hiding/showing elements at different breakpoints (hidden, md:block)
- When adjusting spacing, typography, or grid columns responsively
- When implementing responsive navigation (hamburger menus, collapsible sidebars)
- When working on .cfm view files that need responsive behavior
- When testing or debugging layout issues across screen sizes
- When implementing responsive images or media
- When creating print-friendly layouts with print: prefix

This Skill provides Claude Code with specific guidance on how to adhere to coding standards as they relate to how it should handle frontend responsive design.
