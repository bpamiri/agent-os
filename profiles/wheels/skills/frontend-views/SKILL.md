# Frontend Views

## When to use this skill:

- When creating view files (index.cfm, show.cfm, new.cfm, edit.cfm)
- When implementing layout templates with contentFor() and includeContent()
- When creating reusable partial files (_partial.cfm)
- When working on any .cfm file in views/ directories
- When using cfoutput, cfloop, cfif, and other CFML tags
- When implementing flash message displays
- When building forms with startFormTag()/endFormTag()
- When using view helpers (linkTo, urlFor, textField, select)
- When integrating HTMX attributes (hx-get, hx-post, hx-target)
- When displaying query results or model data in views

This Skill provides Claude Code with specific guidance on how to adhere to coding standards as they relate to how it should handle frontend views.
