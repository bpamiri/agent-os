# Frontend Accessibility

## When to use this skill:

- When adding ARIA labels, roles, and properties to HTML elements
- When implementing keyboard navigation and focus management
- When creating accessible form inputs with proper labels and error messages
- When building navigation menus, dropdown menus, or modal dialogs
- When ensuring sufficient color contrast ratios for text and UI elements
- When adding skip links or landmark regions (main, nav, aside)
- When working on any .cfm view files or HTML templates
- When creating interactive components (tabs, accordions, carousels)
- When writing alt text for images or accessible icon buttons
- When testing with screen readers or accessibility tools

This Skill provides Claude Code with specific guidance on how to adhere to coding standards as they relate to how it should handle frontend accessibility.
