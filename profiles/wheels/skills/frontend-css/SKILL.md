# Frontend CSS

## When to use this skill:

- When adding Tailwind CSS utility classes to HTML elements
- When implementing custom color schemes or theming
- When working on .cfm view files that need styling
- When modifying tailwind.config.js or CSS configuration files
- When implementing spacing, padding, margin utilities
- When adding typography styles (font-size, font-weight, line-height)
- When creating custom CSS in .css or .scss files
- When implementing flexbox or grid layouts with Tailwind
- When adding hover, focus, or other state-based styles
- When working with Tailwind's @apply directive

This Skill provides Claude Code with specific guidance on how to adhere to coding standards as they relate to how it should handle frontend CSS.
