# Global Coding Style

## When to use this skill:

- When writing any code in .cfc, .cfm, .js, or .css files
- When naming variables, functions, components, or files
- When structuring function signatures and method parameters
- When adding comments, whitespace, or line breaks
- When choosing between coding patterns or approaches
- When refactoring existing code for consistency
- When formatting complex conditionals or loops
- When organizing imports or includes at the top of files
- When writing CFML script vs tag syntax
- When reviewing code for style compliance

This Skill provides Claude Code with specific guidance on how to adhere to coding standards as they relate to how it should handle global coding style.
