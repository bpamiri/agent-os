# Global Validation

## When to use this skill:

- When adding model validations (validatesPresenceOf, validatesUniquenessOf)
- When implementing format validations (email, phone, URL patterns)
- When adding length validations (min/max character limits)
- When implementing custom validation methods
- When handling validation error messages in views
- When validating form input in controllers
- When implementing API request validation
- When adding conditional validations (on create, on update)
- When validating file uploads (type, size)
- When implementing cross-field validations (password confirmation)

This Skill provides Claude Code with specific guidance on how to adhere to coding standards as they relate to how it should handle global validation.
