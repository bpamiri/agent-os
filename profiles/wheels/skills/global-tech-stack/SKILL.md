# Global Tech Stack

## When to use this skill:

- When implementing features using CFWheels framework patterns
- When working with Lucee CFML engine specifics
- When integrating HTMX for dynamic page updates
- When using Tailwind CSS and DaisyUI for styling
- When configuring box.json or CommandBox dependencies
- When understanding the project's MVC architecture
- When choosing between available libraries or approaches
- When integrating third-party services or APIs
- When working with the database layer (H2, MySQL, PostgreSQL)
- When understanding deployment and server configuration
